Searching for opportunities to maximise conversion rate when selling XYZ's service to restaurants¶

XYZ company provides staff trainings for restaurants to improve their service quality.

As we all know, restaraunts industry havily relies on customers' reviews on top platforms (Goodle reviews, Yelp, Tripadvisor). Therefore, the first task was to find a dataset with large amounts of customer reviews that we could use for our analysis. There is one great place to start the search from - Kaggle, an online community of data scientists and machine learning practitioners. After spending some time we were able to find and download official Yelp restaurants dataset (5GB).

More facts supporting our incentive for NLP analysis (taken from dataset description):

"Yelp restaurant reviews affected consumers' food choice decision-making; a one-star increase led to 59% increase in revenue of independent restaurants (Luca, 2011)."

We will analyse 5GB dataset from Yelp to find the most criticised areas and suggest XYZ management which restaurants to approach to maximise revenue from selling their trainings. Key objectives:

  1. Clean and aggregate reviews to find most problematic restaurants areas
  2. Perform backet of words segmentation for service-related reviews
  3. Calculate service problems weight coefficient to answer the main question - "Which restaurants to contact to sell XYZ trainings?"

#defining function to load n-rows from 5GB Yelp dataset
def load_rows(filepath, nrows = None):
    with open(filepath) as json_file:
        count = 0
        objs = []
        line = json_file.readline()
        while (nrows is None or count < nrows) and line:
            count += 1
            obj = json.loads(line)
            objs.append(obj)
            line = json_file.readline()
        return pd.DataFrame(objs)

Above we have defined a function to load part of the whole dataset. Otherwise, our PC wouldnt be capable of processing so large amound of data. import matplotlib.pyplot as plt
import seaborn as sns

def FreqDist(words):
    freq = {}
    for word in words:
        if word in freq:
            freq[word]+=1
        else:
            freq[word]=1
    return freq
  
    
# function to plot most frequent terms 
def freq_words(x, terms = 5, graph=True, table=True): 
    
    all_words = ' '.join([text for text in x]) 
    all_words = all_words.split() 
    all_words = [word for word in all_words if word not in ['restaurant']]
    fdist = FreqDist(all_words) 
    words_df = pd.DataFrame({'word':list(fdist.keys()),'count':list(fdist.values())}) 
    
    if graph:
        # selecting top N most frequent words 
        d = words_df.nlargest(columns="count", n = terms)      
        plt.figure(figsize=(20,5)) 
        ax = sns.barplot(data=d, x= d["word"], y = d["count"]) 
        ax.set(ylabel = 'Count') 
        plt.show()
    
    if table:
        return words_df.nlargest(columns="count", n=terms)

2-3. Performing basket analysis and calculating service problems coefficient¶

As we saw from above graphs, there are a lot of problems that restaurants face, but in reality most of the complains can be aggregated under a few categories. Such that:
food, drinks, meat, lettuce, chicken - all related to Food/Kitchen category of problems. 

Therefore, we aggregated similar problems for service and they are: service, staff, time, waiter, manager, server, order, place, minute, line, price, table.
